# Naziv projekta
Web stranica za restoran "Beefsteak Zadar" - prezentacija jelovnika, rezervacija i kontakt informacije.

## Stranice
- index.html - Naslovnica
- about.html - O projektu
- pages/jelovnik.html - Jelovnik
- pages/o-nama.html - Naša priča
- pages/rezervacije.html - Rezervacije
- pages/kontakt.html - Kontakt

## Kako pokrenuti
Otvori index.html u browseru ili koristi Live Server u VS Code.

## Tehnologije
HTML5, CSS3 (Flexbox, Grid, CSS varijable)
